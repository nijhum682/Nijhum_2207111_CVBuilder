# 2207111_java
## Features
- Add personal information including name, contact and address.
- Enter education details for JSC, SSC, HSC and Graduation.
- Add multiple skills, work experiences and projects.
- Upload a photo to include in the CV.
- Clean and organized UI using JavaFX.
- Dynamic CV generation with proper formatting.
